# API Testing Project - Reqres

## Overview
This project demonstrates API testing using Postman.

## Features
- Environment variables (DEV)
- Authentication handling

## How to run
1. Import collection
2. Import environment
3. Run with Postman or Newman

## API used
https://reqres.in

##Test cases
> GET users list
> GET single user
> POST create user
> PUT update user
> DELETE user
> GET user not found
